# 密码管理使用说明

> 🔐 本地加密存储 · Windows DPAPI 保护 · 仅你的账号可访问

---

## 📁 文件位置

- **密码工具：** `C:\Users\bljd5\.openclaw\workspace\password-manager.ps1`
- **密码存储：** `C:\Users\bljd5\.openclaw\workspace\passwords\`

---

## 🚀 使用方法

### 1️⃣ 保存密码

```powershell
cd C:\Users\bljd5\.openclaw\workspace
.\password-manager.ps1 save <服务名> <用户名> <密码>
```

**示例：**
```powershell
.\password-manager.ps1 save github huangjd mypassword123
.\password-manager.ps1 save google huangjd@gmail.com mygooglepass
.\password-manager.ps1 save alipay 13800138000 myalipass
```

---

### 2️⃣ 读取密码

```powershell
.\password-manager.ps1 get <服务名>
```

**示例：**
```powershell
.\password-manager.ps1 get github
# 输出：
# [OK] Service: github
# Username: huangjd
# Password: mypassword123
```

---

### 3️⃣ 查看所有密码

```powershell
.\password-manager.ps1 list
```

**输出示例：**
```
[INFO] Saved passwords:
  - github (User: huangjd, Updated: 2026/3/4 16:18:05)
  - google (User: huangjd@gmail.com, Updated: 2026/3/4 16:19:10)
```

---

### 4️⃣ 删除密码

```powershell
.\password-manager.ps1 delete <服务名>
```

**示例：**
```powershell
.\password-manager.ps1 delete github
```

## 🔒 安全说明

### 加密方式
- 使用 **Windows DPAPI** 加密
- 只有**你的 Windows 账号**能解密
- 密码文件即使被偷也无法在其他电脑读取

### 文件权限
密码文件存储在：
```
C:\Users\bljd5\.openclaw\workspace\passwords\*.txt
```

每个服务一个文件，格式：
```json
{
  "username": "你的用户名",
  "password": "加密后的字符串",
  "time": "保存时间"
}
```

### 安全建议
1. ✅ 不要分享 `passwords` 文件夹
2. ✅ 定期备份密码文件
3. ✅ 使用强密码
4. ⚠️ 不要使用相同密码
5. ⚠️ 重要密码建议用 1Password 等专业工具

## ⚠️ 注意事项

### 优点
- ✅ 免费
- ✅ 无需安装额外软件
- ✅ 完全本地，隐私安全
- ✅ 只有你能访问
- ✅ 可以让我帮你管理

### 缺点
- ❌ 没有跨设备同步
- ❌ 没有手机 App
- ❌ 电脑坏了密码可能丢失
- ❌ 没有密码强度检查
- ❌ 没有自动填充功能

### 建议
- 定期备份 `passwords` 文件夹到安全位置
- 重要密码（银行、邮箱）建议用专业工具
- 日常密码（网站、App）可以用这个管理
