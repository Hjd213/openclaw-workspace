## 🔥 推荐功能 TOP 10

### 1. 网络搜索 ⭐⭐⭐⭐⭐

**用途：** 查资料、搜信息、获取新闻

**状态：** 需要 API key

**配置方法：**
```bash
# 获取 Brave Search API key
# 访问：https://brave.com/search/api/

# 配置到 OpenClaw
openclaw configure --section web
```

**使用示例：**
```
"查一下最新的 AI 新闻"
"搜索 Python 最佳实践"
"看看今天有什么科技大事件"
```

**替代方案：** 让我用浏览器帮你查

---

### 2. GitHub 管理 ⭐⭐⭐⭐⭐

**用途：** 管理 Issues、PRs、代码审查

**状态：** 需要安装 gh CLI

**配置方法：**
```powershell
# 安装 GitHub CLI
winget install GitHub.cli

# 登录
gh auth login

# 验证
gh --version
```

**使用示例：**
```
"查看我的 GitHub Issues"
"帮我创建一个新的 PR"
"检查这个仓库的 CI 状态"
```

---

### 3. 邮件管理 ⭐⭐⭐⭐⭐

**用途：** 收发邮件、管理邮箱

**状态：** 需要配置

**方案 A：Gmail**
- 需要 Google API
- 配置较复杂

**方案 B：IMAP/SMTP（推荐）**
```powershell
# 安装 himalaya
winget install himalaya

# 配置邮箱
himalaya config new
```

**使用示例：**
```
"查看我的未读邮件"
"帮我回复这封邮件"
"发送一封邮件给 XXX"
```

### 6. 语音对话 ⭐⭐⭐⭐⭐

**用途：** 语音输入、语音输出

**状态：** 需要配置

**方案 A：本地语音（免费）**
```powershell

```

- Azure Speech

**使用示例：**
```

```
