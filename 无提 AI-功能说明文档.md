# 无提 AI 功能说明文档

> **产品定位**：桌面智能助手  
> **文档版本**：v1.0  
> **更新日期**：2026-04-11

---

## 一、主界面功能

### 1.1 窗口控制

**用户能看到什么：**
- 一个半透明的悬浮窗口（默认右上角）
- 窗口支持拖拽、缩放
- 最小化到系统托盘

**用户能做什么：**
- **拖拽窗口** → 按住窗口边缘随意移动位置
- **调整大小** → 拖动窗口右下角缩放
- **快捷键呼出** → 按 `Ctrl+Enter` 快速显示/隐藏窗口
- **录屏时隐藏** → 设置中可开启"录屏时自动隐藏窗口"

---

### 1.2 感知开关

**用户能看到什么：**
- 主界面顶部有一个明显的 **感知开关按钮**（带指示灯）
- 绿色 = 感知开启中
- 灰色 = 感知已关闭

**用户能做什么：**
- **点击开关** → 一键开启/关闭屏幕和语音感知
- **查看状态** → 指示灯实时显示当前状态
- **快捷键切换** → 按 `Ctrl+Shift+P` 快速切换感知状态

---

### 1.3 模型切换

**用户能看到什么：**
- 设置页面有一个 **模型选择下拉框**
- 显示当前可用模型列表（云端 + 本地）
- 每个模型旁边标注：在线/离线、速度、准确度

**用户能做什么：**
- **切换模型** → 下拉选择：
  - Gemini（云端，快，准）
  - Qwen（云端，快，中等）
  - ChatGPT（云端，中等，准）
  - 本地模型（离线，慢，中等）
- **下载本地模型** → 点击"下载本地模型"按钮：
  - 显示模型列表（7B、13B、70B 参数）
  - 显示下载进度条
  - 下载完成后自动显示"已安装"
- **删除本地模型** → 已安装的模型旁边有"删除"按钮

---

## 二、感知功能

### 2.1 屏幕内容感知

**用户能看到什么：**
- 设置页面有一个 **屏幕感知开关**
- 实时预览窗口（可选开启）：显示当前 AI"看到"的屏幕内容

**用户能做什么：**
- **开启/关闭屏幕感知** → 开关控制
- **选择感知范围**：
  - 仅当前窗口
  - 整个屏幕
  - 指定应用（如只感知浏览器）
- **上传文件** → 点击"上传文件"按钮，选择 PDF/Word/Excel/PPT

---

### 2.2 语音感知

**用户能看到什么：**
- 设置页面有一个 **麦克风开关**
- 音量指示条（实时显示输入音量）
- 语言选择下拉框（中文/英文/日文...）

**用户能做什么：**
- **开启/关闭麦克风** → 开关控制
- **选择语言** → 下拉选择主要使用的语言
- **测试麦克风** → 点击"测试"按钮，说话看音量条是否跳动
- **查看转录历史** → 点击"历史记录"查看过去的语音转录内容

---

### 2.3 主动建议

**用户能看到什么：**
- 设置页面有一个 **主动建议开关**
- 灵敏度滑块（低/中/高）
- 建议历史记录（过去 7 天的主动建议）

**用户能做什么：**
- **开启/关闭主动建议** → 开关控制
- **调节灵敏度**：
  - 低 = 很少主动建议，几乎不打扰
  - 中 = 适度建议，每天几次
  - 高 = 频繁建议，随时提供建议
- **反馈建议质量** → 每条建议旁边有 👍/👎 按钮
- **屏蔽场景** → 可以设置"在游戏时不要建议"、"在视频播放时不要建议"

---

## 三、交互功能

### 3.1 对话界面

**用户能看到什么：**
- 输入框（支持语音和文字）
- 历史对话列表（左侧）
- 当前上下文预览（右侧，显示 AI 看到的屏幕内容摘要）

**用户能做什么：**
- **文字提问** → 在输入框打字，按 Enter 发送
- **语音提问** → 点击麦克风图标，说话后自动发送
- **引用上下文** → 点击"引用当前屏幕"，AI 基于当前屏幕内容回答
- **上传文件提问** → 拖拽文件到输入框，针对文件内容提问
- **追问** → 在同一个对话中连续提问，AI 记得上下文
- **删除对话** → 鼠标悬停在对话上，点击删除图标

---

### 3.2 快捷操作

**用户能看到什么：**
- 设置页面有一个 **快捷键配置表**
- 每个功能旁边显示当前快捷键

**用户能做什么：**
- **修改快捷键** → 点击快捷键，按新的组合键：
  - 呼出窗口：`Ctrl+Enter`（可改）
  - 切换感知：`Ctrl+Shift+P`（可改）
  - 语音输入：`Ctrl+Shift+V`（可改）
  - 截图提问：`Ctrl+Shift+S`（可改）
- **恢复默认** → 点击"恢复默认快捷键"按钮

---

## 四、场景功能

### 4.1 会议模式

**用户能看到什么：**
- 主界面有一个 **会议模式按钮**
- 开启后显示：
  - 实时转录文字（滚动显示）
  - 参会人数（如可识别）
  - 会议时长
  - 结束会议按钮

**用户能做什么：**
- **开启会议** → 点击"开始会议"按钮
- **结束会议** → 点击"结束会议"，自动生成会议纪要
- **查看纪要** → 会议结束后自动弹出纪要，支持编辑
- **导出纪要** → 点击"导出"按钮，选择 Word/Markdown/飞书文档
- **历史会议** → 在"会议记录"页面查看所有历史会议

---

### 4.2 场景模板

**用户能看到什么：**
- 设置页面有一个 **场景选择器**
- 显示可用场景卡片：
  - 🏥 医疗场景
  - 💰 金融场景
  - 💼 销售场景
  - 📚 学习场景
  - ⚙️ 通用场景

**用户能做什么：**
- **选择场景** → 点击场景卡片切换
- **查看场景说明** → 鼠标悬停显示该场景的特殊功能
- **自定义场景** → 点击"创建自定义场景"，设置专属 prompt 模板

---

### 4.3 医疗场景（示例）

**用户能看到什么：**
- 切换到医疗场景后，界面显示医疗相关快捷操作：
  - 病历分析
  - 药品查询
  - 诊断建议
  - 合规检查

**用户能做什么：**
- **病历分析** → 点击按钮，AI 自动分析当前屏幕上的病历内容
- **药品查询** → 输入药品名，查询相互作用、副作用
- **诊断建议** → 基于症状描述，提供可能的诊断方向（仅供参考）
- **合规检查** → 检查病历书写是否符合规范

---

### 4.4 金融场景（示例）

**用户能看到什么：**
- 切换到金融场景后，界面显示金融相关快捷操作：
  - 风险提示
  - 竞品对比
  - 数据溯源
  - 合规检查

**用户能做什么：**
- **风险提示** → AI 自动识别当前对话中的风险点并提示
- **竞品对比** → 输入产品名，自动生成竞品对比表格
- **数据溯源** → 点击任意数据，显示来源和可信度
- **合规检查** → 检查话术是否符合金融监管要求

---

## 五、自动化功能

### 5.1 文件整理

**用户能看到什么：**
- 自动化页面有一个 **文件整理功能**
- 显示要整理的文件夹路径
- 预览整理后的效果

**用户能做什么：**
- **选择文件夹** → 点击"选择文件夹"按钮
- **选择整理规则**：
  - 按文件类型分类（图片/文档/视频）
  - 按日期分类（2026/04/2026/03...）
  - 按项目分类（基于文件名识别）
- **预览效果** → 点击"预览"查看整理后的结构
- **确认执行** → 点击"执行"按钮，确认后开始整理
- **撤销操作** → 整理后 24 小时内可点击"撤销"恢复原状

---

### 5.2 格式转换

**用户能看到什么：**
- 自动化页面有一个 **格式转换功能**
- 显示支持转换的格式：
  - PDF ↔ Word
  - Word ↔ PDF
  - Excel ↔ CSV
  - 图片 → 文字（OCR）

**用户能做什么：**
- **选择文件** → 拖拽文件到转换区域
- **选择目标格式** → 下拉选择要转换的格式
- **批量转换** → 可选择多个文件一起转换
- **开始转换** → 点击"转换"按钮
- **下载结果** → 转换完成后点击"下载"

---

### 5.3 文本提取

**用户能看到什么：**
- 自动化页面有一个 **文本提取功能**
- 显示支持的格式：PDF、图片、扫描件

**用户能做什么：**
- **选择文件** → 拖拽文件到提取区域
- **提取文本** → 点击"提取"按钮
- **复制结果** → 点击"复制"按钮
- **导出文件** → 点击"导出"保存为 TXT/Word

---

## 九、系统要求

| 要求 | 说明 |
|-----|------|
| 操作系统 | Windows 10/11 或 macOS 12+ |
| 内存 | 最低 4GB，推荐 8GB+ |
| 磁盘空间 | 基础安装 500MB，本地模型额外需要 2-10GB |
| 网络 | 云端模型需要联网，本地模型可离线使用 |
| 权限 | 屏幕录制权限、麦克风权限、文件访问权限 |
