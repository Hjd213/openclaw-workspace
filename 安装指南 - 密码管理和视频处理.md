## 1️⃣ 安装 1Password（密码管理）

### 方案 A：使用 1Password 服务（推荐）

**步骤：**

1. **注册 1Password 账号**
   - 访问：https://1password.com
   - 点击 "Try 1Password Free"
   - 注册免费试用账号（14 天试用，之后$3/月）

2. **下载 1Password 应用**
   - 访问：https://1password.com/downloads/windows
   - 下载并安装 Windows 版

3. **安装 1Password CLI**
   - 访问：https://developer.1password.com/docs/cli/
   - 下载 Windows 版 CLI
   - 或运行：
   ```powershell
   winget install 1Password.CLI
   ```

4. **登录 CLI**
   ```powershell
   op signin
   ```
   按提示输入：
   - 账号邮箱
   - 密钥（Secret Key，注册时邮件发送）
   - 主密码

5. **验证安装**
   ```powershell
   op --version
   # 应显示版本号
   ```

### 方案 B：使用本地密码文件（免费替代）

如果不想用 1Password，我可以帮你管理加密的密码文件。

**步骤：**

1. **创建密码文件**

## 2️⃣ 安装 FFmpeg（视频处理）

### Windows 安装

**方法一：使用 winget（推荐）**

```powershell
# 以管理员身份打开 PowerShell
winget install Gyan.FFmpeg --source winget
```

**方法二：手动安装**

1. **下载 FFmpeg**
   - 访问：https://www.gyan.dev/ffmpeg/builds/
   - 下载 "ffmpeg-release-essentials.zip"

2. **解压**
   - 解压到：`C:\Program Files\FFmpeg`

3. **添加到环境变量**
   - 右键"此电脑" → 属性 → 高级系统设置
   - 环境变量 → 系统变量 → Path → 编辑
   - 新建：`C:\Program Files\FFmpeg\bin`
   - 确定保存

4. **验证安装**
   ```powershell
   # 打开新的 PowerShell 窗口
   ffmpeg -version
   # 应显示版本信息
   ```

### 验证成功标志

```powershell
ffmpeg -version
# 输出应包含 ffmpeg version 字样
```

## 3️⃣ 安装 codexbar（模型用量统计）

### 安装步骤

```powershell
# 以管理员身份打开 PowerShell
npm install -g codexbar

# 验证安装
codexbar --version
```

### 配置 API Key

```powershell
# 设置 OpenAI API Key（如果你有）
codexbar config set openai_api_key YOUR_API_KEY

# 或者设置阿里云 API Key
codexbar config set bailian_api_key YOUR_API_KEY
```

### 使用示例

```powershell
# 查看用量
codexbar cost

# 查看今日用量
codexbar cost --today

# 查看本月用量
codexbar cost --month
```

## 🔧 故障排查

### 问题 1：命令找不到

**原因：** 环境变量未生效

**解决：**
1. 关闭所有 PowerShell 窗口
2. 重新打开 PowerShell
3. 再次尝试验证命令

### 问题 2：winget 安装失败

**原因：** winget 未安装或版本过旧

**解决：**
```powershell
# 更新 winget
winget install winget

# 或手动下载安装包
# 访问：https://apps.microsoft.com/detail/9NBLGGH4NNS1
```

### 问题 3：npm 安装失败

**原因：** Node.js 未安装或权限不足

**解决：**
```powershell
# 以管理员身份运行
# 或使用 --force 参数
npm install -g codexbar --force
```
