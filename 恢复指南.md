# 从零恢复 OpenClaw 配置指南

> 换设备/重装系统后，按这个流程恢复你的 AI 助手 🐾

---

## 前置条件

- Windows 10/11
- Node.js 20+ ([下载地址](https://nodejs.org/))
- Git ([下载地址](https://git-scm.com/))
- GitHub 账号

---

## 步骤 1: 克隆仓库

```powershell
# 创建文件夹
mkdir C:\Users\你的用户名\.openclaw
cd C:\Users\你的用户名\.openclaw

# 克隆仓库
git clone https://github.com/Hjd213/openclaw-workspace workspace
cd workspace
```

---

## 步骤 2: 安装 OpenClaw

```powershell
# 全局安装 OpenClaw
npm install -g openclaw
```

---

## 步骤 3: 配置 GitHub（用于代码工作）

```powershell
# 登录 GitHub
gh auth login

# 按提示操作：
# 1. 选择 GitHub.com
# 2. 选 HTTPS
# 3. 选 Login with web browser
# 4. 复制代码，浏览器打开授权
```

**配置 Git 用户信息：**

```powershell
git config --global user.name "你的 GitHub 用户名"
git config --global user.email "你的邮箱"
```

---

## 步骤 4: 配置模型 API

1. 打开 `C:\Users\你的用户名\.openclaw\openclaw.json`
2. 填入你的 API Key（阿里云百炼/其他提供商）

**或者运行：**

```powershell
openclaw config
```

按提示填入 API Key。

---

## 步骤 5: 启动 OpenClaw

```powershell
# 启动 Gateway
openclaw gateway start

# 查看状态
openclaw status
```

---

## 步骤 6: 验证恢复

检查以下文件是否存在：

- `C:\Users\你的用户名\.openclaw\workspace\SOUL.md` ✅
- `C:\Users\你的用户名\.openclaw\workspace\IDENTITY.md` ✅
- `C:\Users\你的用户名\.openclaw\workspace\USER.md` ✅
- `C:\Users\你的用户名\.openclaw\workspace\memory\` ✅

如果都在，说明恢复成功！🎉

---

## 常见问题

### Q: `openclaw` 命令找不到？

A: 确保 Node.js 已安装，并且 npm 全局路径在系统 PATH 中。

### Q: GitHub 推送失败？

A: 网络问题，多试几次，或者用网页上传。

### Q: 记忆文件丢失？

A: 检查 `memory/` 文件夹是否随仓库一起克隆下来。
