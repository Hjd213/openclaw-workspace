## 🚀 一、快速开始

### 这是什么？

OpenClaw 是一个**个人 AI 助手框架**，可以：
- 7×24 小时待命，主动提醒你事情
- 帮你搜索信息、整理笔记、自动化任务
- 连接你的各种服务（待办、笔记、浏览器等）
- 像真人一样跟你聊天，不是冷冰冰的机器人

### 核心特点

| 特点 | 说明 |
|------|------|
| **有记忆** | 记得你说过的话、做过的事，不会聊完就忘 |
| **主动提醒** | 每天定时检查邮件、日历、天气，有重要事情会主动找你 |
| **能干活** | 不只是聊天，能真正帮你完成任务（搜索、写文档、自动化） |
| **可定制** | 性格、说话风格、工作内容都能按你的喜好调整 |
| **本地优先** | 数据存在你自己电脑上，不依赖云端 |

## 💬 四、日常使用

### 怎么跟 AI 聊天？

**当前渠道**：WebChat（浏览器界面）

**说话风格**：随意、直接、不废话
- ❌ "您好，请问有什么可以帮助您的吗？"
- ✅ "有啥事？"

**可以叫它**：Claw、小助手、任意称呼

### 常用指令

```
# 查询状态
/status              # 查看会话状态、用量

# 管理记忆
/memory search <关键词>   # 搜索记忆
/memory get <路径>        # 读取记忆文件

# 定时任务
/cron list           # 查看所有定时任务
/cron run <任务 ID>   # 手动执行任务

# GitHub 操作
/gh issues           # 查看 GitHub issues
/gh prs              # 查看 Pull Requests
```

### 让它干活

**搜索整理类**：
```
"帮我搜一下最近 3 天 AI 领域的重大新闻，整理成笔记"
"查一下明天上海的天气，要不要带伞"
```

**自动化类**：
```
"把我桌面截图里的文字提取出来"
"把这个视频的关键帧提取出来"
"把我的待办事项同步到 Obsidian"
```

**创作类**：

## ⏰ 五、定时任务（心跳 & Cron）

### 心跳检查（Heartbeat）

**频率**：每天 2-3 次（不固定时间）

**当前配置**（`HEARTBEAT.md`）：
- ✅ 自动检查 workspace 更改
- ✅ 自动 git commit + push
- ✅ 推送成功/失败都通知用户

**可以添加的检查项**：
```markdown
## 我的待办检查
- 查看 Todoist 今天到期的任务
- 如果有紧急任务，提醒我

## 邮件检查
- 检查 Gmail 未读邮件
- 有重要邮件就通知我

## 日历提醒
- 检查未来 24 小时的事件
- 提前 1 小时提醒我
```

### Cron 定时任务

**当前任务**：
| 任务 | 时间 | 说明 |
|------|------|------|
| AI 资讯晨报 | 每天 8:00 | 收集 AI 新闻并发送 |

**添加新任务示例**：
```

```

## 🔐 六、安全与隐私

### 它能做什么（安全）

- ✅ 读取你的文件、笔记、配置
- ✅ 搜索网络信息
- ✅ 在你的 workspace 里创建/修改文件
- ✅ 执行 git 操作
- ✅ 调用已配置的 API（Obsidian、Todoist 等）

### 需要确认才能做

- ❓ 发送邮件、推文等公开内容
- ❓ 删除文件（默认用 `trash` 而非 `rm`）
- ❓ 任何可能影响外部的操作
- ❓ 访问你没配置过的服务

### 隐私保护

- **MEMORY.md 只在私聊时读取**，群聊时不会泄露你的个人信息
- **敏感数据**（API Key、密码）放在 `TOOLS.md`，不要发到群里
- **群聊时**：AI 只是参与者，不会代替你发言

## 🛠️ 七、故障排查

### 常见问题

**Q: AI 不回复了？**
```
1. 检查 openclaw gateway 状态：openclaw gateway status
2. 重启：openclaw gateway restart
3. 查看日志找错误
```

**Q: 定时任务没执行？**
```
1. /cron list 查看任务状态
2. 检查 lastRunStatus 是不是 error
3. 看 lastError 字段找原因
```

**Q: git push 失败？**
```
1. 检查网络连接（可能需要代理）
2. 手动执行：cd workspace && git push
3. 还是不行就用 GitHub 网页上传
```

**Q: AI 记不住我说过的话？**
```
1. 确保在 main session（私聊）
2. 检查 memory/ 目录是否有每日文件
3. 重要的事让它写进 MEMORY.md
```

## 📚 八、进阶玩法

### 1. 自定义 AI 性格

编辑 `SOUL.md`，例如：
```markdown
## Vibe
- 说话像好朋友，不要太正式
- 可以开玩笑，但不要过度
- 专业问题时认真，闲聊时随意
```

### 2. 添加个人技能

创建 `skills/` 目录，放自定义脚本：
```
skills/
├── my-automation/
│   ├── SKILL.md      # 技能说明
│   └── script.py     # 你的脚本
```

### 3. 连接更多服务

参考 `TOOLS.md` 添加：
- 智能家居（Home Assistant）
- 摄像头监控
- SSH 远程服务器
- 自定义 API

### 4. 自动化工作流

用 Cron + 技能组合：
```
每天早上 7 点：
1. 查天气 → 决定穿什么
2. 查日历 → 提醒今天的事
3. 查邮件 → 汇总重要邮件
4. 一次性汇报给我
```

## 🎯 九、给新手的建议

### 第一天做什么？

1. **改 USER.md**：告诉 AI 你是谁、怎么称呼你
2. **改 SOUL.md**：设定你喜欢的说话风格
3. **试几个命令**：让它搜索、总结、创建笔记
4. **配置一个定时任务**：比如每天早上提醒你看待办

### 第一周做什么？

1. **连接你的笔记软件**（Obsidian/Notion）
2. **连接待办软件**（Todoist/Things）
3. **设置自动备份**（GitHub/Gitee）
4. **尝试自动化**：让它帮你做一件重复的事

### 第一个月做什么？

1. **整理 MEMORY.md**：把重要的偏好、决定写进去
2. **添加更多技能**：根据需求安装新技能
3. **优化定时任务**：调整心跳检查的频率和内容
4. **分享给朋友**：让他们也用起来
