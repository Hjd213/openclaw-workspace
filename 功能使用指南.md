## 🔐 功能一：密码管理

### 使用方法

**打开 PowerShell，进入工作目录：**
```powershell
cd C:\Users\bljd5\.openclaw\workspace
```

### 1. 保存密码

```powershell
.\password-manager.ps1 save <服务名> <用户名> <密码>
```

**示例：**
```powershell
.\password-manager.ps1 save github huangjd MyPass123
.\password-manager.ps1 save google huangjd@gmail.com GooglePass
```

### 2. 读取密码

```powershell
.\password-manager.ps1 get <服务名>
```

**示例：**
```powershell
.\password-manager.ps1 get github
```

### 3. 查看所有密码

```powershell
.\password-manager.ps1 list
```

### 4. 删除密码

```powershell
.\password-manager.ps1 delete <服务名>
```

### 🔒 安全说明

- 密码使用 Windows DPAPI 加密
- 只有你的 Windows 账号能解密
- 密码文件存储在 `workspace\passwords\` 目录
- 不要分享这个文件夹

## 🎬 功能二：视频处理

### 前置要求

✅ FFmpeg 8.0.1 已安装  
路径：`C:\Users\bljd5\AppData\Local\Microsoft\WinGet\Links\ffmpeg.exe`

### 使用方法

**打开 PowerShell，进入工作目录：**
```powershell
cd C:\Users\bljd5\.openclaw\workspace
```

### 1. 查看视频信息

```powershell
.\video-tools.ps1 info <视频文件>
```

**示例：**
```powershell
.\video-tools.ps1 info C:\Videos\myvideo.mp4
```

### 2. 提取视频帧

```powershell
.\video-tools.ps1 frames <视频文件> <输出目录>
```

**示例：**
```powershell
.\video-tools.ps1 frames video.mp4 frames
# 每秒提取一帧，保存到 frames 文件夹
```

### 3. 裁剪视频

```powershell
.\video-tools.ps1 trim <输入> <输出> <开始时间> <时长>
```

**示例：**
```powershell
.\video-tools.ps1 trim video.mp4 clip.mp4 00:00:10 00:00:30
# 从 10 秒开始，裁剪 30 秒
```

### 4. 创建 GIF

```powershell
.\video-tools.ps1 gif <输入> <输出> <开始时间> <时长>
```

**示例：**
```powershell
.\video-tools.ps1 gif video.mp4 out.gif 00:00:10 5
# 从 10 秒开始，创建 5 秒的 GIF
```

### 5. 转换视频格式

```powershell
.\video-tools.ps1 convert <输入> <输出>
```

**示例：**
```powershell
.\video-tools.ps1 convert input.avi output.mp4
```

## 💡 使用技巧

### 密码管理

1. **服务名命名建议**
   - 使用小写：`github`, `google`, `alipay`
   - 可以加子域名：`github.work`, `github.personal`

2. **定期备份**
   ```powershell
   # 复制 passwords 文件夹到安全位置
   Copy-Item .\passwords D:\Backup\passwords-$(Get-Date -Format yyyyMMdd) -Recurse
   ```

### 视频处理

1. **批量提取帧**
   ```powershell
   # 每 5 秒提取一帧
   .\video-tools.ps1 frames video.mp4 frames
   # 然后手动删除不需要的帧
   ```

2. **高质量 GIF**
   ```powershell
   # 提高 FPS 和分辨率
   .\video-tools.ps1 gif video.mp4 out.gif 00:00:10 5
   # 编辑 video-tools.ps1 修改 fps=15,scale=1280:-1
   ```

### 模型用量

1. **定期查看**
   - 每周运行一次 `.\usage.ps1`
   - 监控 Token 使用量
   - 估算费用，避免超支

2. **导出数据**
   ```powershell
   .\model-usage.ps1 export
   # 数据保存在 ~/.openclaw/usage-stats.json
   ```
